
# Terminal chess
![Screenshot](./screenshot.png)

## Compile and run
```sh
make
./chess
```
## What is implemented
- [x] Display board
  - [x] rudimentary letters
  - [x] colored tiles with unicode symbols
- [x] Parse user input
  - [x] rudimentary parsing
  - [ ] algebraic notation
  - [ ] detect notation
- [ ] Move validation
- [ ] Check for winner
- [ ] Log game score sheet
